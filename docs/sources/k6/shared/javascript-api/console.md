---
title: javascript-api/console
---

The `console` object provides methods for logging messages during test execution. It is available globally without requiring an import statement.

| Method                                                                               | Description                    |
| ------------------------------------------------------------------------------------ | ------------------------------ |
| [console.log()](https://grafana.com/docs/k6/<K6_VERSION>/javascript-api/console/log) | Logs a message to the console. |
