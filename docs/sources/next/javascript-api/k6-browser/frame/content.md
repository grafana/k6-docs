# content()

Gets the HTML content of the frame.

### Returns

| Type              | Description                                                                                                             |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------- |
| `Promise<string>` | A Promise that fulfills with the HTML content of the frame as a string value or `null` if the attribute is not present. |
